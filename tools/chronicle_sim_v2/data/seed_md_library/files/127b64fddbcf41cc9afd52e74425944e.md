# 川渝骂人话

## 概述

川渝地区（重庆、四川）的骂人话有其独特的历史和文化背景。清末民初的重庆是码头城市，江湖气息浓厚，街上说话带脏字是常态，不脏反而不像本地人。

**核心特点：**
- 脏话常以**歇后语**形式出现，只说前半句，听者自补后半句
- 自称"老子"是江湖标配，不是真的骂人，是语气助词
- 语气词"锤子""铲铲"表示否定、不屑，日常使用频率极高

---

## 常用词

| 词汇 | 含义 | 使用场景 |
|------|------|---------|
| **妈卖批** | 国骂级别，极度愤怒 | 类似"他妈的"，码头工人、脚夫日常用语 |
| **日你妈** | 最直接的骂人话 | 真急眼了才用，一般不随便说 |
| **哈儿** | 傻子、蠢货 | 日常骂人，程度不重，熟人之间常用 |
| **芽儿** | 嫩、不成熟、毛都没长齐 | 老人骂年轻人、老手骂新手时用 |
| **瓜娃子** | 傻孩子 | 程度轻，带点亲昵的骂法 |
| **龟儿子** | 没出息、孬种 | 通用骂人，程度较轻 |
| **仙人板板** | 骂祖宗 | 极度愤怒时 |
| **锤子** | 否定词，"才不是" | "你懂个锤子" |
| **铲铲** | 同锤子 | "有个铲铲用" |
| **老子** | 我（傲慢版） | 日常自称 |
| **格老子** | 老子+语气助词 | 重庆特有感叹词 |

## 动词类

| 词汇 | 含义 | 示例 |
|------|------|------|
| **日决** | 骂、训斥 | "他被老子日决了一顿" |

| **冲壳子** | 吹牛 | 同"日白" |
| **踩假水** | 说假话 | 袍哥黑话 |

## 歇后语

| 前半句 | 后半句 | 含义 |
|--------|--------|------|
| 宝批龙进茶馆 | 大不同 | 骂人装模作样 |
| 又哭又笑 | 黄狗飙尿 | 嘲笑人情绪不稳定 |
| 茅房里头跳高 | 奋（粪）勇前进 | 讽刺人瞎积极 |
| 耗子舔猫屁屁 | 找死 | 骂人不自量力 |

---

## 特殊用法：只说半句

川渝人骂人习惯只说前半句，后半句听者自己补。

| 只说半句 | 完整意思 |
|---------|---------|
| "你娃就是个……" | 瓜娃子/哈儿 |
| "老子今天非……" | 收拾你/日决你 |
| "我看你是……" | 哈儿 |
| "仙人……" | 板板 |

## 与游戏关联

### 关二狗的说话方式

关二狗是个底层小人物，在茶馆"打滚"混日子：

- **日常**："锤子""哈儿""老子"
- **对袍哥**：不敢直接骂，心里嘀咕"龟儿子"
- **对脚帮**：平级互骂，"你个哈儿""瓜兮兮的"
- **生气时**："妈卖批""仙人板板"
- **被老辈人骂**：被说"芽儿，毛都没长齐"

### NPC说话示例

| NPC | 风格 | 示例 |
|-----|------|------|
| 茶馆舵爷 | 表面客气实际敲打 | "你娃莫在这儿冲壳子哦" |
| 脚帮老辫子 | 粗犷直接 | "哈儿，醒都醒了还躺起干啥子" |
| 瞎子李 | 老人对年轻人的嗔怪 | "你个芽儿，做事毛躁得很" |
| 普通百姓 | 嫌弃害怕 | "妈卖批，又是袍哥的人" |
