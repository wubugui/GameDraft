# (b) 占位素材阶段 光影/镜头调参 —— 用近零成本把"廉价感 + 撞脸怀旧RPG"一起压掉

> 对照真实实现：`src/rendering/lightEnv.ts`（三层合并 BASELINE < game_config.defaultLightEnv < scene.lightEnv）、`game_config.json#entityLighting`、`public/assets/data/filters/*.json`（色彩矩阵）。
> 已有系统：逐实体光照 + 投影阴影(real) + 深度遮挡 + 场景色彩矩阵滤镜。**不需要新代码**，全是填字段。

## 0. 现状诊断（这是问题根源，也是最便宜的胜负手）

- **当前没有任何一个场景写 `lightEnv`，也没场景挂 filter。** 实测：`rg lightEnv/filter public/assets/scenes` = 0 命中。
- 于是**所有场景都吃 `game_config.json#entityLighting.defaultLightEnv` 这套亮调**：
  `key` 暖白 `[1.0,0.97,0.92]` intensity **1.0**、`ambient` 冷灰 `[0.55,0.6,0.72]` intensity **1.0**、`shadow.darkness` **0.45**、`toneStrength` **0.45**。
- 后果：城隍庙夜、阎王岭山口、枯井、义庄、灵堂、梦_夜路 **全在用"白天均匀亮光"渲染** → 这正是"像怀旧仙剑白天 / 换皮页游"的来源。
- **撞脸的真凶不是"45°像仙剑"（合体是有意的），而是"亮平的占位画面像怀旧RPG白天"。** 修它不用换素材，只要给每个场景塞 `lightEnv` + 一道色彩分级。

## 1. 两个独立的杠杆（必须分清，都要用）

| 杠杆 | 作用对象 | 怎么用 | 解决什么 |
|---|---|---|---|
| **场景滤镜（色彩矩阵）** | **整个画面，含占位背景图** | `filters/*.json` 4×5 矩阵给场景分级 | 把"亮平占位背景"整体压暗、去饱和、染冷/染暖——**去廉价感最狠的一刀** |
| **scene.lightEnv** | **实体（角色/道具精灵）的受光与投影** | 每个场景写 lightEnv 块 | 让人物**坐进**场景：戏剧布光 + 长投影 + 冷暖对立——出"志怪"气质 |

> 背景是预画好的占位图，lightEnv 主要打实体；想压暗**背景本身**，靠滤镜色彩矩阵。两者叠加才完整。

## 2. 总原则（反"均匀明亮水墨" = 反怀旧）

1. **让阴影成为主角**：默认画面是"黑里抠细节"，不是"亮堂堂展示场景"。
2. **单点/低角度光**：弃"全景均匀照亮"，用油灯/月光/手电式的方向感强光。
3. **冷暖对立**：暗部压冷、点光压暖，制造体积与不安，而不是整体一个色。
4. **统一色调分级**：每个场景一套 grade，画面"脏、潮、旧"，把占位素材的"干净 GPT 味"盖掉。
5. **暖光要留给"安全/欺骗"**：开场茶馆故意暖亮让玩家放松，后面骤变（uncanny）。

## 3. 可调字段速查（全部来自 lightEnv.ts，可直接写进 scene.lightEnv）

| 字段 | 含义 | 调暗/调恐怖方向 |
|---|---|---|
| `ambient.intensity` | 环境光强度（**总亮度主开关**） | **最重要**：从 1.0 砍到 0.2–0.4 = 全场变暗 |
| `ambient.color` | 暗部色 | 夜=冷蓝 `[0.28,0.34,0.5]`；灯火=暗暖 `[0.18,0.16,0.18]` |
| `key.intensity` | 主光强度 | 0.5–0.7（弱主光，靠对比不靠亮） |
| `key.color` | 主光色 | 月光冷 `[0.55,0.65,1.0]`；油灯暖 `[1.0,0.78,0.45]` |
| `key.elevationDeg` | 主光仰角（**越低投影越长**，length=cot，夹[0.3,1.6]） | 12–22 = 长拖影；恐怖必备 |
| `key.azimuthDeg` | 光来向（0=右，逆时针） | 侧后方来光（如 200–250）= 脸暗、轮廓亮、最瘆 |
| `shadow.darkness` | 投影浓度 | 0.75–0.88（深） |
| `shadow.softness`/`softSamples`/`softRadius` | 软阴影 | 软一点更阴翳（softSamples 3–6, softRadius 0.08–0.14） |
| `shadow.drape` | 阴影贴地/披挂 | 0.6–0.8 |
| `toneStrength` | 色调融入强度（ambient 色染整体） | 0.6–0.8（强 grade） |
| `ao.contact`/`ao.form` | 接触/体积 AO | 0.5–0.7（暗角更实） |

## 5. 全局色彩分级（压占位背景的廉价感）

现有 `filters/night.json` 已是"R/G 砍到 0.4、B 抬到 0.7+0.1"的冷暗矩阵——这套思路就对。建议**给恐怖场景统一挂一道更"志怪"的冷暗低饱和 grade**（4×5 颜色矩阵，行=RGBA 输出，列=R,G,B,A,偏移）：

```json
{
  "id": "zhiguai_night",
  "matrix": [
    0.34, 0.05, 0.03, 0, 0,
    0.04, 0.34, 0.05, 0, 0,
    0.06, 0.08, 0.46, 0, 0.04,
    0,    0,    0,    1, 0
  ],
  "alpha": 1
}
```
- 对角线压到 0.34/0.34/0.46 = 整体变暗、蓝偏冷；非对角小量交叉 = 轻微去饱和（把 GPT 占位的高饱和"干净感"压成"旧、脏、潮"）。
- 白天阴翳场景用一套更轻的（对角 0.7 左右、略去饱和）。
- **接线方式同 night.json**：在滤镜编辑器里建这张表，按场景挂上（与你现有 filter 应用路径一致）。

## 6. 镜头/构图（低成本反撞脸）

1. **第一印象资产定生死**：商店头图 / 宣传片首帧 / 开场 30 秒，**必须暗调光影 + 志怪符号 + 音效**定调，**绝不要用明亮全景地图截图当门面**——这是切断"这是仙侠RPG"误读最关键的一下。
2. **别用仙剑式"全景看清整张地图"**：镜头略拉近、压低，让画面边缘吃进暗部（配 §5 的 grade + 暗 ambient，四角自然变黑）= 廉价版"视野收束"。
3. **真·Darkwood 视野锥（视野锥+战争迷雾）是 L2 工作**：需要新增"以角色为中心的可见度遮罩"渲染（现有 lightEnv/filter 做不到）。**占位阶段先用"暗 ambient + 冷暗 grade + 单点暖光 + 暗角"近似**；要不要上真视野锥，等 demo 验证完"俯视也能瘆人"再决定（符合你"最多+1新系统"的范围纪律）。
4. **长拖影是你已实现系统的免费红利**：`key.elevationDeg` 压到 12–22，逐实体投影自动拉长——人物在月光下拖一条长影走过空场，瘆感直接拉满，零美术成本。

## 8. 验收：「第一眼测试」

每个场景出一张首帧截图，问一句：**"这像志怪历险，还是像怀旧仙侠RPG白天 / 换皮页游？"**
- 若答案偏后者 → `ambient.intensity` 再砍、`toneStrength` 再加、grade 再压。
- 通过线：占位素材在暗调 + 冷暖对立 + 长投影下，**第一眼读作"民国志怪夜路"而不是"晴天古风地图"**。
- 这一关过了，你最担心的"撞脸"和品类最致命的"廉价小作坊感"，**在没换任何一张素材的前提下**就压掉了一大半。
