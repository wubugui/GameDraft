# 《寻狗记》重庆话语料库

> **用途**：为民国（清末民初）题材等距冒险游戏《寻狗记》写角色对白时的逐条查用手册。设定地＝渝都卫／雾津＝**重庆**一带（码头、义庄、袍哥世道）。主角关二狗（码头扒尸混混、粗俗文盲）、李天狗（半文游方道士）、市井NPC、说书人等。
> 写对白时**先查本表**：确认词条属「重庆通用／成渝通用」可放心用；属「成都特有」的进避雷表、改用对应重庆词；属「存疑」的少用、按备注降级。

---

## 总则：重庆话 ≠ 成都／四川话

成渝方言**高度共通**（《汉语方言词汇》同质率约 92.5%），所以绝大多数词标「成渝通用」即可，不必当成都词排除。真正要守住「重庆味」的，是下面这几条**高判别度分水岭**——一旦写错就立刻暴露成都腔：

| 维度 | 重庆（用这个） | 成都（避开） |
|---|---|---|
| 疑问语气词「吗」 | **迈**（你吃了迈？）/ 直接「…没得？」 | 哇（你吃了哇？）—**头号避雷** |
| 「没有」 | **没得** | 莫得 |
| 「怎么」 | **啷个** | 咋个 |
| 「这里／那里」 | **勒点／那点**（勒里） | 这儿/那儿、这里/那里 |
| 「这个」 | **勒个** | 这个 |
| 「这么／那么」 | **恁个**（浪个） | 这们/那们、这样子 |
| 程度「很」 | **嘿X**（嘿沉、嘿巴适，前置） | ~得很/~惨了（后置） |
| 骂「傻」 | **哈儿／宝器** | 瓜娃子／瓜兮兮 |
| 「加油」 | **雄起**（重庆原生） | （加油） |
| 「楼下」 | 楼脚 | 楼下 |

**全程铁律（北方腔红线）**：禁用 您／俺／咱们／哩，禁儿化卷舌（「一会儿」写「一哈／这哈」，不写「一哈儿」）；叹词不用北方「哎哟喂」，改用「哦豁／哟／嗨呀」。尊称代偿：成渝都无「您」，对长辈/陌生人靠**称呼**（师傅/老板儿/先生/老者/哥老倌/老师）＋语气表敬，不用敬称代词。

### 用户钦定词表（最高优先，别给相反建议）

| 普通话 | 一律写作 | 说明 |
|---|---|---|
| 尸首／尸体／尸身 | **死人子** | 角色台词统一用，不用「尸体/尸首」；构词（X+子）符合西南官话习惯，比「尸体」有码头扒尸的粗野市井气，贴关二狗身份 |
| 昨晚／昨天晚上 | **昨晚上** | 配套：今晚＝今天黑了 |
| 句末「噻」（肯定/劝说） | **撒** | 音同噻/唦，**绝非疑问**，别和迈/嗦混 |
| 句末「喃」（疑问/自问） | **葛** | 按音写，本音 lán/nan，＝「呢」 |
| 丢／掉 | **落**（没掉＝没落） | 与「没得＝没有」分工不冲突：没得管「没有」、没落管「没掉」 |

## 一、人称·称谓

| 重庆词 | 普通话义 | 例句 | register | 备注 |
|---|---|---|---|---|
| 我 | 我（中性自称） | 我晓得了。 | 中性/口语 | 发音近 ngo/o，不需刻意写方言字；粗俗加重才上「老子」。关二狗中性叙述用「我」即可 |
| 你 | 你 | 你啷个还在这点？ | 中性/口语 | 禁「您」；对长辈也用「你」，靠称呼/语气表尊重 |
| 你娃 / 你娃儿 | 你这家伙、你小子 | 你娃少给老子来这套。 | 口语/粗俗 | 损人或熟人挑衅、教训口吻，比单说「你」更冲，适合关二狗 |
| 你龟儿 / 你龟儿子 | 你这龟儿、你他妈 | 你龟儿跑哪点去了嘛！ | 粗俗/江湖 | 码头/江湖高频口头禅，怒骂或哥们打趣都用；重庆骂人「冲祖上」，这类带亲属脏字最对味。关二狗主力粗话 |
| 他 / 他娃 | 他；他那家伙（轻蔑） | 他娃龟儿子又开溜了。 | 中性至粗俗 | 「他」中性，带情绪说「他娃/他龟儿」 |
| 别个 | 别人、人家、那个人 | 别个又没惹你，凶啥子嘛。 | 中性/口语 | 川渝通用高频，替普通话「人家/别人」，市井味足 |
| 我们 / 我们这些 | 我们、咱们 | 我们码头上的人，哪个怕过他。 | 中性/口语 | 不用北方「咱们」；复数用「我们/你们/他们」「我们这些/你们那些」 |
| 崽儿 | 小子、男娃（带不恭）；本地土生男子 | 这个崽儿胆子倒是肥。 | 口语/粗俗 | 重庆色彩明显（「重庆崽儿」本地标志称呼），适合混混互称 |
| 婆娘 | 女人（不恭）；老婆 | 那个婆娘嘴巴凶得很。 | 口语/粗俗 | 贬称女人或粗指老婆，骂蠢女人作「瓜婆娘」 |
| 堂客 | 老婆、妻子 | 他堂客在码头卖茶。 | 口语 | 下力人/市井口语称妻子，年代感强，比「老婆」土。成都标志词是「老妞儿」——勿用 |
| 老汉 / 老汉儿 | 父亲、爸爸 | 我老汉早死了，没得人管我。 | 口语 | 口语指爸爸；与骂人语境分开，单说指父亲 |
| 老者 | 老头子、老人家 | 那个老者怕是有七十喽。 | 口语 | 称年长男性，市井年代味；当面客气叫「老人家」 |
| 婆婆 | 祖母；老年妇女 | 门口那个婆婆天天在卖针线。 | 中性/口语 | 重庆称祖母作「婆婆/婆子妈」，泛称老妇亦可 |
| 师傅 | 对有手艺/出力者、陌生男性的尊称 | 师傅，搭把手嘛！ | 中性/口语 | 码头下力人圈最常用，代替不存在的「您」。关二狗求人、搭讪用 |
| 老板儿 | 老板、掌柜 | 老板儿，行行好，赏口饭撒。 | 口语 | 重庆口语爱加儿尾，称店家/抬举对方。注意：此「儿」是词尾非卷舌儿化（非北方音），可保留；怕混淆可写「老板」 |
| 先生 | 对读书人/有身份者/算命/教书/郎中的尊称 | 先生，与我看个字撒。 | 中性/书场 | 民国通用尊称；关二狗抬举李天狗、账房、教书匠时用，书面感比「师傅」强 |
| 哥老倌 / 老倌 | 袍哥/江湖弟兄间对成年男子的称呼 | 哥老倌些，赏个脸撒。 | 江湖 | 哥老会江湖称呼，民国川渝码头帮会色彩浓，正对袍哥面向 |
| 老师 | 对陌生人的通用尊称（无关教书） | 老师，借问一声，赶尸的从哪条路走？ | 口语 | 重庆对陌生人统一尊称，问路搭话用，年代亦通 |
| 告花子 / 叫花子 | 乞丐 | 码头边一排告花子，伸手讨钱。 | 口语/年代 | 年代市井群像词 |

## 二、自称与骂詈（粗俗·关二狗主力区）

| 重庆词 | 普通话义 | 例句 | register | 备注 |
|---|---|---|---|---|
| 老子 | 我（粗俗自大男性自称，「你爹」口气） | 老子今天就要落你这碗饭！ | 粗俗/江湖 | 仅男性用，气愤/逞凶/撑场面/哥们打趣时上。关二狗被惹毛时核心自称；不宜对真正长辈/上位者乱用 |
| 格老子 | 粗口感叹/发狠语（约「我操/我的妈呀」） | 格老子，这死人子咋个恁个沉！ | 粗俗/江湖 | 句首发语感叹，吃惊/恼火/发狠起头。有来源称本含敬畏（「我的天」），市井多作粗口发语，给关二狗按粗口用 |
| 锤子 | 屁、个屁、放屁、才怪（强烈否定/蔑视） | 帮你？锤子哦！ | 粗俗/江湖 | **重庆标志性粗口**，重庆比成都用得更凶更随口，强重庆味，可多用。本义为工具锤子，骂用为引申 |
| 哈儿 | 傻子、傻瓜 | 你个哈儿，连这都看不出来。 | 粗俗 | 重庆骂傻首选（成都偏「瓜娃子」）；重庆色彩重 |
| 哈批 / 哈麻批 | 傻子（加脏版） | —— | 粗俗 | 带「批」为加重脏话，更冲 |
| 宝器 / 宝气 / 宝批龙 | 傻冒、爱出洋相又没本事；不上台面 | 你个宝器，死人子身上的东西也敢拿？ | 粗俗 | 重庆高频骂人词，**重庆比成都更当真**、当面叫易翻脸。「宝批龙/宝兮兮」同源。注意：「宝气」也指娇气/小气，按语境区分 |
| 龟儿 / 龟儿子 | 王八蛋；口头亲昵脏字 | 龟儿子，落了老子的钱还想跑！ | 粗俗/江湖 | 川渝通用脏话，码头江湖口头禅，怒骂或打趣都用，常与第二人称连用「你龟儿」。关二狗主力脏字 |
| 莽子 / 莽娃 | 傻大个、缺心眼的莽汉 | 那个莽子力气大，脑壳不灵光。 | 粗俗 | 骂憨/愣/缺心眼；「莽」也作形容词（莽起做＝闷头硬干） |
| 神戳戳 / 神兮兮 / 神抖抖 | 神经兮兮、举止反常犯傻 | 对到死人子自言自语，神戳戳的。 | 口语/粗俗 | ABB 状态词，成渝通用，悬疑/诡异氛围贴；骂「傻/疯癫」的重庆主力仍是「哈儿」 |
| 瓜兮兮 / 瓜 | 傻乎乎、呆 | 莫站到瓜兮兮的，过来搭把手！ | 口语 | 成渝都说，但带成都色彩；要纯重庆味换「宝器」 |
| 戳锅漏 | 成事不足败事有余的人 | 莫喊那个戳锅漏来搭手，越帮越乱。 | 口语/贬抑 | 年代市井贬人词 |
| 背时 | 倒霉、走背运；轻骂「该死的、挨刀的」 | 背时哦，又遇到他！／背时砍脑壳的！ | 口语 | ＝倒霉，也作轻骂，年代市井骂咧好用。配钦定「昨晚上」：昨晚上背时遭官差查 |
| 涮坛子 | 开玩笑、戏耍人、忽悠 | 道士莫涮老子的坛子，正经说价。 | 口语/江湖 | 江湖味玩笑词 |
| 扯把子 | 撒谎、吹牛 | 道士先生莫跟我扯把子，你那符到底灵不灵。 | 口语 | 市井常用 |
| 说聊斋 | 扯皮、瞎扯、说没影的话 | 莫跟我说聊斋，钱货两清。 | 口语 | 年代市井词，文盲关二狗说「聊斋」反有反差趣味 |

**关二狗骂人风格定调（重庆式）**：短、冲、劈头盖脸、五分钟完事、嗓门大、**冲对方家属和祖上**（龟儿/老子）。**不**走成都那套绵软幽默、绕弯子的描述性长损（牙尖/占明占眼那种酸味长损）。

## 三、句末语气词与叹词

| 重庆词 | 普通话义 | 例句 | register | 备注 |
|---|---|---|---|---|
| 撒（噻） | 肯定/劝说/理所当然（噻+吧+嘛合体，把话说软） | 走撒，莫站起打望了。／就是这个理撒。 | 口语 | **钦定写法**：句末「噻」一律写「撒」。最通用的肯定/劝说语气词。**绝非疑问**，别和迈/嗦混 |
| 嘛 | 理所当然/显而易见/催促/停顿（「这不明摆着」） | 吃嘛喝嘛，莫客气。／拿都拿来了，背起就走嘛。 | 口语 | 带「万事皆小事」的放任语气，邀请/催促/认命都用。关二狗这种混混常用，撇脱 |
| 迈 | 疑问「吗」（是非问） | 你吃了迈？／这死人子是你扒的迈？ | 口语 | **重庆专用、对成都「哇」**。本面向只用迈、不用哇。也可不用语气词直接说「你吃没得？」 |
| 哈 | 叮嘱/求认同/缓和命令（「啊/好吧/记住了」） | 你给我记到哈。／慢点走哈，码头滑。 | 口语 | 祈使句尾加哈，整句变柔软；李天狗这种客气口吻很合适 |
| 嗦 | 疑问「是…吗」带轻蔑/戏谑/明知故问（降调） | 你龟儿瓜的嗦？／钱不想要了嗦？ | 口语 | 多半心里已有定论，常含轻蔑/调侃，挑衅味重。混混斗嘴、江湖口角合用 |
| 葛（喃） | 疑问「呢」（追问/自问/反诘） | 你说咋个办葛？／我落了的那把刀葛？ | 口语 | **钦定写法**：句末「喃」一律写「葛」。本音 lán/nan |
| 哟 | 惊讶/招呼/起头打量（哟/嚯） | 哟，大嫂，你也来了嗦！／哟，这死人子还热和的。 | 口语 | 句首叹词起头招呼/惊讶，句末也可拖一个哟。市井气足 |
| 哦 | 升调表恍然/确认「原来如此」；降调表应答「晓得了」 | 哦——原来是你龟儿搞的鬼。／哦，要得，我晓得了。 | 口语 | 成渝同 |
| 哦豁 | 事情砸了/糟了/没指望了（失望、惋惜、无奈） | 哦豁，跑脱了！／哦豁，钱落进河头了。 | 口语 | **极高频重庆叹词**，「有一种失望叫哦豁」。出事/落空/搞砸瞬间脱口，比普通话「完了/糟了」地道得多。本游戏出岔子最地道反应 |
| 嗨呀 / 哎呀 | 引出解释或感叹（无奈/惊讶/夸张铺垫） | 嗨呀，你莫提了，今天背了一整天的死人子。 | 口语 | 起头叹词引出抱怨/解释。别带北方「哎哟」儿化或「俺」味 |
| 嘿（句首叹词） | 起头招呼/引人注意/加强语气（嘿/喂） | 嘿，那个背篓的，站到起！ | 口语 | 句首叹词喝住人，江湖喝问味重，适合码头混混。区别于副词「嘿」＝很（靠位置分） |

## 四、高频动词·形容词·副词

| 重庆词 | 普通话义 | 例句 | register | 备注 |
|---|---|---|---|---|
| 晓得 | 知道、懂 | 你龟儿晓得个铲铲！这死人子昨晚上才落的水。 | 口语 | 西南官话核心词，否定「不晓得」。比「知道」地道。李天狗、关二狗都用 |
| 要得 | 好、行、可以（应答） | ——帮我把这具死人子拖上岸。——要得，钱先讲清楚。 | 口语 | 重庆口里硬朗；否定/反问作「要不得」（不行/使不得） |
| 没得 | 没有 | 身上没得一个铜板，还想叫我背死人子？ | 口语 | **对成都「莫得」，成渝分水岭**。「没有」一律用没得；与钦定「没落＝没掉」分工不冲突 |
| 嘿（黑） | 很、非常（程度，前置） | 这具死人子嘿沉，一个人背不动。 | 口语 | **重庆最标志性程度副词**，「很」读 hēi。「嘿巴适/嘿吓人」。成都偏「~得很」，重庆「嘿X」频率显著高，辨味关键词 |
| 得很 | （形容词后）很、得不得了 | 码头边臭得很，死人子泡胀了。 | 口语 | 成渝皆通（成都尤甚）。与「嘿X」可互换；突出重庆味优先「嘿X」 |
| 焦（焦干/焦苦/焦黄） | 极、非常（前置，多形容坏的极端） | 晒了三天，那张老脸焦黄焦黄的。 | 口语 | 重庆特色程度前缀，多修饰负面状态（焦苦/焦干/焦湿）。年代码头戏可用 |
| 稀（稀烂/稀脏/稀溜） | 极、非常（程度前缀） | 那双草鞋遭水泡得稀烂。 | 口语 | 西南官话广用，重庆口里常听 |
| 安逸 | 舒服、爽、满意；反讽时指惨/糟 | 睡了个安稳觉，安逸惨咯。 | 口语 | 成渝都重用，**重庆比「巴适」更倾向说这个**，可放心给重庆角色。多形容人的主观感受，可反讽（冷安逸了＝冷惨了） |
| 造孽 | 可怜、作孽、遭罪（西南官话特征义，非北方「作恶」） | 这娃儿爹妈都死绝了，造孽哦。 | 口语 | 常配「造孽哦」，年代悲悯戏好用 |
| 活路 | 活计、工作、营生、生计 | 扒死人子也是门活路，总比饿肚子强。 | 口语 | **关二狗身份本命词**，下力人语境极贴（棒棒「一根棒棒一条活路」）。「找活路/做活路」 |
| 下力 / 下力人 | 出卖力气干粗活（的人）、苦力 | 我们这些下力的，命就是力气换饭。 | 口语/年代 | 民国码头阶层自称，年代感正 |
| 将就 | 凑合、勉强、迁就 | 没得好的，这口薄棺材你将就用。 | 中性 | 川渝口语使用频率远高，日常高频 |

**存疑（少用/降级，勿断言为重庆独有）**：
- **嘎/噶**（征求附和）：成都材料更常举为招牌词，重庆听得懂但更典型用迈/嗦/葛。列存疑、慎用；需「征求附和」用撒/哈替代。
- **勺子**（傻子）：多归为「四川话」未细分城市，重庆能听到但非最重庆味。关二狗优先哈儿/宝器/锤子，勺子作备选少用。
- **资格**（正宗/地道）：川渝口语确有（资格货/耍得资格），但「重庆 vs 成都」归属无明确切分。可用但勿断言重庆独有，要稳妥优先安逸/正宗。
- **哪阵**（什么时候）：西南官话有「哪阵/那阵/这阵」，重庆能听到但更偏川西/部分区县，主城更常说「好久」。建议李天狗这类半文角色偶用。
- **死人子**：钦定最高优先、直接采用。网上未检索到独立词条，属角色/项目设定用语；构词（X+子）符合西南官话习惯，标注为设定词。

### 关二狗（码头扒尸混混·粗俗文盲）
- **自称/口头**：老子、格老子（发狠起头）；中性叙述用「我」。
- **脏字主力**：龟儿/龟儿子、锤子（否定）、哈儿/宝器（骂傻）、你龟儿、你娃。
- **风格**：短、冲、劈头盖脸、冲祖上；遇事「啷个搞嘛」「哦豁」「遭不住」。讨价用「好多钱」「相因」「撇脱」「钱货两清」。
- **身份词**：活路、下力、搭手、整、切、刹一脚。
- **句末**：撒（劝/肯定）、嘛（认命/催促）、迈（问）、嗦（轻蔑反问）、葛（追问）。
- **绝不**：文气客套、绕弯子长损（牙尖式）；绝不用 您/俺/咱们/儿化；绝不用 哇/咋个/莫得/巴适/瓜娃子/啦。

### 李天狗（半文游方道士）
- **该用**：莫（莫慌/莫提了）、晓得、要得、哈（缓和叮嘱：慢点走哈）、先生（互称尊称）、造孽（悲悯）、将就。半文角色可偶用「哪阵」「之乎」式书面词稍抬。
- **风格**：客气、把话说软（多用「哈/撒」收尾），**不带粗口**——不说锤子/龟儿/老子。
- **绝不**：码头江湖脏话密集化；不用 您/俺/儿化；问句用迈不用哇。

### 市井NPC（茶贩、店家、街坊、围观群众）
- **该用**：师傅/老板儿/老者/婆婆/老师（称呼代敬）、晓得、要得、没得、嘛、撒、哟、哦豁、打望、安逸、造孽。
- **打气场面**：雄起撒！
- **风格**：生活气、年代名物（灶屋/堂屋/茅厮/棒棒/滑竿）；闲话八卦带「别个」「打望」。
- **绝不**：成都标志词（哇/巴适/老妞儿/楼下）；儿化/北方腔。

### 说书人（书场腔）
- **该用**：先生/哥老倌（书场称呼）、年代隐语全套（袍哥/舵把子/拜码头/吃讲茶/拉稀摆带/落教/认黄/光棍）、地理彩蛋「通远门看埋死人」「朝天门」。
- **风格**：半文半白、抑扬顿挫，可上谚语歇后语、城门民谣；register 偏「年代/书场」。
- **绝不**：现代词、普通话直译；不用 您/俺/儿化。
