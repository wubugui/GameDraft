# 人民主体与反商品化玩法设计

> 本文整理一次关于“人民领袖 / 人民主体 / 防止资本逻辑复辟”的创作讨论，并将其转译为《GameDraft》可落地的世界观、叙事、玩法与系统设计方向。
>
> 设计目标不是把思想写成口号，而是让玩家通过民俗解谜、传闻辨析、社会关系修复与非战斗式对抗，亲手体验：一个地方如何被“万物标价”的逻辑腐蚀，又如何靠共同记忆、公道、禁忌与群众互助重新站起来。

---

## 1. 核心命题

不要把主题直接命名为“反资本主义”，而是转译为：

> 夺回人的位置。

在游戏世界中，外来的商号、买办、军阀、官绅、洋行、矿主、票号，不一定以“恶人”面目出现。他们更危险的地方在于：他们把人、土地、尸骨、神灵、传闻、记忆、婚丧嫁娶、乡土关系全部变成可以定价、抵押、买卖和宣传的东西。

玩家要做的不是杀死某个老板，而是让人重新成为人，让地方重新有自己的魂。

---

## 2. 主题转译原则

### 2.1 不说教

避免让 NPC 反复说抽象政治口号。思想必须进入以下层面：

- 世界规则；
- 解谜逻辑；
- NPC 关系；
- 传闻传播；
- 地图变化；
- 任务反馈；
- 玩家可操作的“词语 / 知识 / 禁忌”。

玩家不是听懂理论，而是在局中看见：

- 谁被孤立；
- 谁被标价；
- 谁被迫沉默；
- 谁通过传闻获利；
- 哪些民俗禁忌其实在保护共同体边界；
- 哪些“理性发展”正在把人从地方关系里拆出来。

### 2.2 不把反派写成脸谱恶人

最好的反派不是嘴上喊“我要剥削人民”的坏人，而是体面、理性、效率至上的人。

他可能修桥、办学、赈灾、开工厂，甚至讲契约、讲发展、讲秩序。但他做的一切都在把地方原本的非商品关系拆掉：

- 师徒变雇佣；
- 乡邻变债务人；
- 神明变景观；
- 茶馆变宣传口；
- 婚姻变抵押；
- 尸骨变矿权；
- 传闻变市场工具；
- 地名变编号；
- 人名变账目。

反派代表的不是“坏”，而是一种逻辑：

> 世界上没有神圣之物，只有可定价之物。

---

## 3. 世界观表达

### 3.1 渝都卫的社会结构

渝都卫可以被设计成一个原本依靠民间网络维持秩序的城市：

- 茶馆负责传话、评理、说书；
- 城隍庙负责公共道德与地方裁判；
- 脚帮熟悉地下水脉、旧路、坟地和城外险路；
- 手艺人维持器物、仪式和生活秩序；
- 乡约与禁忌约束人与土地、亡者、神灵之间的边界；
- 庙会、渡口、井、桥、祠堂，是公共生活节点。

外部资本化力量进入后，不一定马上破坏城市，而是逐步“盘活”它：

- 老井开始收费；
- 庙产被改造成商馆；
- 荒坟被划入矿权；
- 婚丧仪式被债务化；
- 传闻被商号操控；
- 茶馆说书被收买；
- 地名被改成工程编号；
- 人的信誉被账本替代。

### 3.2 民俗禁忌不是装饰，而是边界

民俗禁忌可以被解释为老百姓保护共同体的古老边界，而不是单纯迷信。

例子：

- “祖坟不能动”：保护土地记忆、家族根系和亡者尊严；
- “井口不能卖”：保护公共水源；
- “城隍庙不能改作商铺”：保护公共裁判和道德空间；
- “新丧之家不能逼债”：保护弱者最低尊严；
- “孕妇房中不动土”：保护脆弱生命不受施工、争吵、搬迁扰动；
- “桥头不能设赌局”：保护来往行人不被诱入债务；
- “夜里不能喊人全名”：保护人的身份不被邪术和债契捕捉。

当反派无视禁忌、强行拆改、定价、抵押时，诡异事件发生。鬼不一定主动害人，鬼更像是被压下的旧账显形。

## 4. 核心系统：传闻不是提示，而是社会现实

### 4.1 传闻系统定位

传闻不是普通任务线索，而是一种会改变社会现实的力量。

传闻具备以下属性：

- 来源：谁最早说出；
- 传播者：茶馆、码头、庙会、赌档、商号伙计、脚帮、媒婆；
- 受害者：谁被孤立、污名化、驱赶、逼债；
- 获利者：谁因此得到土地、名声、债权、工程、矿权；
- 旧说法：是否存在被压住的民间版本；
- 证据链：账本、契约、旧歌谣、碑文、口供、地名、禁忌；
- 反转方式：不是辟谣一句话，而是恢复关系与证词网络。

### 4.2 传闻示例

#### 传闻 A：东街寡妇命硬，克死丈夫

表层：寡妇不祥，街坊避让。

实际：商号想低价收她家的地，先制造社会孤立，让她失去求助网络。

玩法：

- 查丈夫死因；
- 找脚帮确认当晚渡口记录；
- 从茶馆说书段子中发现污名化话术；
- 找出伪造债契；
- 恢复街坊证词。

主题：人不能被传闻剥夺社会身份。

#### 传闻 B：矿主修桥是积德

表层：矿主出钱修桥，百姓感恩。

实际：修桥是为了改道运矿，同时掩盖上游水源污染和船工死亡。

玩法：

- 观察河道变化；
- 查旧水文歌谣；
- 询问船工遗属；
- 对照工程账本；
- 解开桥头禁忌。

主题：发展叙事可能遮蔽剥夺。

#### 传闻 C：脚帮都是盗墓贼

表层：脚帮不可信，见不得光。

实际：脚帮掌握地下旧路、水脉和坟地真相，因此被商号污名化。

玩法：

- 通过脚帮暗语进入旧城地下；
- 证明他们曾经救过灾、修过堤；
- 让脚帮重新获得发言位置。

主题：被污名化的人可能保存着地方真实记忆。

#### 传闻 D：欠债还钱，天经地义

表层：债务人不守信用。

实际：债来自伪造契约、利滚利、官府勾结与灾年逼签。

玩法：

- 找到账本原件；
- 破解票号暗码；
- 找到旧乡约中关于灾年债务的限制；
- 让群众共同见证债契无效。

主题：契约不一定天然正义，必须追问契约产生的条件。

## 5. 核心系统：词语 / 知识转化为工具

玩家收集的不是普通钥匙，而是一些民间词、伦理词、革命词、地方规矩。这些词可以成为“思想工具”。

### 5.1 词语工具示例

#### 公道

作用：让 NPC 愿意说出被压下的证词。

玩法表现：当“公道”被激活，某些原本沉默的 NPC 会补全关键证词，但前提是玩家已经恢复其安全感或社会支持。

#### 乡约

作用：恢复社区互助网络。

玩法表现：开启多人协作解谜，例如街坊共同开门、共同作证、共同拒签新契。

#### 账本

作用：看穿商号、官府、庙产、票号之间的隐秘交易。

玩法表现：把多个案子的资金流、人物关系、地契变化连接成一张大图。

#### 根

作用：让被商品化的土地显现原本历史。

玩法表现：地图恢复旧地名、旧路、旧井、旧坟、旧碑文。

#### 香火

作用：凝聚共同记忆，不是迷信值。

玩法表现：香火恢复后，庙不再只是场景，而成为公共证词和群众集结点。

#### 翻身

作用：解除 NPC 对自身低贱命运的认同。

玩法表现：被压迫 NPC 从“我命该如此”转为“我也能说话 / 作证 / 拒绝”。

#### 不是买卖

作用：破解把人情、婚姻、尸体、神灵标价的邪术。

玩法表现：解除阴婚契、尸骨买卖、庙产抵押等仪式化束缚。

### 5.2 词语不是按钮，而是关系钥匙

每个词语必须通过剧情理解后才能使用。不能变成 UI 技能按钮。

建议结构：

1. 玩家第一次听到词语，只是文本；
2. 玩家经历一个事件，理解词语背后的关系；
3. 词语变成工具；
4. 工具不是对物体生效，而是对社会关系生效；
5. 使用后改变 NPC、地图、传闻、禁忌或证词网络。

## 6. 核心数值：香火 / 公道 / 人心

避免使用传统 RPG 的战斗力、装备、金币作为核心成长。建议建立三个社会状态值。

### 6.1 香火

含义：共同记忆、地方精神、民间信任。

下降表现：

- 庙变成收费景点；
- 祭祀变成生意；
- 老人不愿讲旧事；
- 地名消失；
- 鬼影变得混乱和攻击性更强。

上升表现：

- 旧歌谣恢复；
- 庙祝打开正门；
- 鬼影从害人变成指路；
- 旧地名重新出现在地图上。

### 6.2 公道

含义：真相能否被说出，弱者是否有人听。

下降表现：

- NPC 不愿作证；
- 契约压倒事实；
- 官府只认账本；
- 茶馆只讲对商号有利的故事。

上升表现：

- 证词链出现；
- 伪造契约暴露；
- 茶馆说书改口；
- 群众共同见证。

### 6.3 人心

含义：社区互助程度，普通人是否敢站出来。

下降表现：

- 邻里互相举报；
- 遇事只问“对我有什么好处”；
- 弱者被孤立；
- NPC 只提供交易，不提供帮助。

上升表现：

- 街坊主动提供线索；
- 脚帮帮玩家开旧路；
- 孩子唱回被篡改的歌谣；
- 群众拒签新契。

---

## 7. 玩家定位

玩家不应该是单人救世主。

如果玩家一个人查案、破局、拯救所有人，本质上仍然是英雄史观。为了表达“人民主体”，玩家的作用应该是：

- 发现被孤立的人；
- 恢复被切断的关系；
- 让沉默者重新说话；
- 让错误传闻被纠正；
- 让旧规矩重新被理解；
- 让大家一起完成某个行动。

章节结尾尽量避免“玩家打败反派”，而应呈现群众重新形成共同体：

- 茶馆说书人改口；
- 脚帮愿意作证；
- 寡妇拿出旧契；
- 船工敲响码头铜锣；
- 庙祝打开城隍庙正门；
- 街坊共同拒绝签新契；
- 孩子们把被篡改的歌谣重新唱回来。

玩家是点火的人，真正改变局面的是群众重新连接起来。

## 8. 章节设计草案

### 第一章：卖井

商号买下一口老井，开始按桶收钱。

传闻说井里有水鬼，必须由商号镇压管理。

真相：商号污染上游水源，再制造恐慌垄断水。

核心机制：

- 通过水质、旧井碑、卖水账、夜间鬼影查真相；
- 用“井口不能卖”的禁忌恢复公共水源属性；
- 最终由街坊共同拆掉收费木牌。

主题：公共资源不能私有化。

### 第二章：阴婚契

穷人家女儿死后，被富户买去配阴婚。

传闻说这是“给她找归宿”。

真相：债务、身体、婚姻、死亡都被变成交易。

核心机制：

- 查媒婆话术、债契、亡者遗物；
- 用“不是买卖”破解阴婚契；
- 让亡者不再作为债务抵押物出现。

主题：人不是商品，尸骨也不是商品。

### 第三章：神仙顶矿梦

有人梦见神仙顶下有金矿，大量人被诱去挖山。

传闻说这是神仙赐财。

真相：矿主利用民间传说制造淘金狂热，破坏龙脉和村落水系。

核心机制：

- 对照梦境、地脉图、脚帮旧路、水脉歌谣；
- 发现所谓“神仙梦”来自刻意投放的符纸与说书段子；
- 玩家不是阻止发财，而是让群众看见发财神话背后的代价。

主题：发财神话如何吞噬共同体。

### 第四章：城隍庙改商馆

城隍庙被改成商会会馆。

传闻说旧神无用，新商能兴城。

真相：地方失去公共裁判后，契约、债务、谣言开始无人约束。

核心机制：

- 找回城隍庙旧案卷；
- 恢复庙前公议场；
- 让多个章节的证词汇聚到庙中。

主题：公共道德空间被资本占领后，公道会失声。

### 第五章：铁算盘的账

最终发现每个案子都在一张大账里。

不是某个鬼在作祟，而是整座城被改造成一部账本机器。

人名、地名、坟名、庙名、河名都被换成编号和价格。

最终解法不是杀死反派，而是让全城人重新说出旧名、旧事、旧债、旧冤。

主题：人民重新夺回命名权和历史记忆。

## 9. 关键反派原型：铁算盘

暂定名：沈铁算盘 / 陆铁算盘 / 陆掌柜。

角色定位：不是暴力恶人，而是“万物标价”的代理人。

常用话术：

- “荒坟也是地。”
- “庙产闲置，不如盘活。”
- “穷人借钱，总要付利。”
- “死人占着好地，活人怎么发展？”
- “传闻也是生意，恐惧也是流量。”
- “人情最不可靠，契约才公平。”
- “这不是抢，是交易。”
- “这不是逼，是选择。”
- “这不是卖祖宗，是发展地方。”

他的可怕之处：

- 他永远显得体面；
- 他善于用理性词汇掩盖剥夺；
- 他能让受害者觉得自己活该；
- 他能把群众拆成孤立个体；
- 他不怕鬼神，因为他只认账。

最终对抗不应靠杀死他，而应靠账本失效：当群众重新形成共同记忆与共同见证，他的契约、传闻和定价逻辑无法继续运转。

---

## 10. 终局意象

最终反派说：

> 你们以为自己有祖宗、有乡土、有神灵、有公道。可到最后，不都要写进账本？

玩家最后不是拔剑，而是让之前收集的词语、证词、歌谣、禁忌、旧名、旧账、旧冤汇合：

- 井不是货；
- 庙不是铺；
- 尸不是物；
- 债不是命；
- 人不是价；
- 城不是你的；
- 这个地方，是大家的。

终局表达：

> 这个城，是大家的。

这句话可以作为对“这个国家是你们的”的游戏化转译，但不要直接照搬现实政治表达。

## 11. 可落地的系统任务

### 11.1 传闻图谱系统

目标：建立一个可编辑的数据结构，用来表达传闻、来源、传播者、受害者、获利者、证据、反制方式。

建议字段：

```json
{
  "rumorId": "well_ghost_001",
  "title": "老井有水鬼",
  "surfaceClaim": "老井闹鬼，必须由商号镇压管理",
  "hiddenInterest": "商号污染上游水源并垄断卖水",
  "sourceNpc": "商号伙计",
  "spreadChannels": ["茶馆", "码头", "庙会"],
  "victims": ["东街居民", "挑水人"],
  "beneficiaries": ["恒昌商号"],
  "evidence": ["旧井碑", "卖水账", "上游水样", "船工证词"],
  "counterWords": ["公道", "井口不能卖"],
  "resolvedWorldChanges": ["井口收费牌移除", "街坊恢复互助", "水鬼变为指路鬼影"]
}
```

### 11.2 词语工具系统

目标：让“词语”作为解谜工具，而不是普通物品。

建议字段：

```json
{
  "wordId": "gongdao",
  "displayName": "公道",
  "unlockCondition": "完成茶馆证词链",
  "meaning": "让被压下的事实重新被听见",
  "validTargets": ["沉默NPC", "伪造契约", "公共见证场景"],
  "effects": ["NPC愿意作证", "证词图谱解锁", "公道值上升"]
}
```

### 11.3 社会状态系统

目标：香火 / 公道 / 人心影响地图、NPC、传闻和结局，不作为战斗数值。

建议字段：

```json
{
  "areaId": "east_street",
  "xianghuo": 35,
  "gongdao": 20,
  "renxin": 42,
  "thresholdEvents": [
    {
      "condition": "gongdao >= 50",
      "event": "茶馆说书人改口"
    },
    {
      "condition": "renxin >= 60",
      "event": "街坊共同拒签新契"
    }
  ]
}
```

## 12. 创作边界

本设计涉及现实政治思想的抽象转译，但游戏中建议保持架空表达：

- 不直接使用现实人物与真实政治事件；
- 不复刻真实仪式、真实亡者、真实咒语；
- 不把民俗简化为迷信或猎奇；
- 不把穷人写成天然善良、富人写成天然邪恶；
- 不把“人民主体”写成主角救世；
- 不把思想落成口号，而要落成选择、关系、地图变化与群众行动。

## 13. 一句话设计总结

> 把资本逻辑写成一种“万物标价”的邪术；把民俗禁忌写成共同体保护自己的古老边界；把玩家写成重新连接人民、记忆和公道的人；最终让整座城从账本里逃出来。
