# 《寻狗记》重庆话语料库

> **用途**：为民国（清末民初）题材等距冒险游戏《寻狗记》写角色对白时的逐条查用手册。设定地＝渝都卫／雾津＝**重庆**一带（码头、义庄、袍哥世道）。主角关二狗（码头扒尸混混、粗俗文盲）、李天狗（半文游方道士）、市井NPC、说书人等。
> 写对白时**先查本表**：确认词条属「重庆通用／成渝通用」可放心用；属「成都特有」的进避雷表、改用对应重庆词；属「存疑」的少用。
> 来源：5 面向上网查证（见文末来源）＋ 项目内活范本（用户亲笔·码头水猴子戏）校准。**冲突时以本节「校准修正」为准。**

---

## ★ 校准修正（最高优先，覆盖下文任何冲突）

研究稿与「用户亲笔活范本」（见文末§项目内活范本）对账后的硬修正：

1. **「咋个／咋子」不排除**。研究稿把它划成都特有、要全换「啷个」——**过度了**。用户本人台词用「咋子嘛」「咋没见过」。定为：**啷个** 最具重庆辨识度、做首选；**咋个/咋子** 成渝都用、**不禁**（关二狗、NPC 都可说）。避雷表里的「咋个」相应降级为「优先啷个」而非「排除」。
2. **「得很」有效**。研究稿据「嘿X前置」把后置「得很」降级——**过度了**。用户用「哈得很」「凶得很」。定为：**嘿X**（嘿沉/嘿臭，前置）最重庆味、优先；**得很**（后置）同样地道可用，二者并存。
3. **补两个句末语气词**（研究稿漏，用户在用）：
   - **也**：句末软化/陈述（用户「我前几天还下水洗过澡**也**」）。
   - **索/嗦**：明知故问/带轻蔑反问（用户「你还当真了**索**？」＝研究稿的「嗦」，用户写「索」，二字皆可，**优先按用户写「索」**）。
4. 核查三修：①**瓜/瓜兮兮**只进避雷表（重庆骂傻用 哈儿/宝器），不进可用正表；②**雄起** 标「重庆标志喊法」而非「重庆原生」（起源成渝有争议）；③**格老子** 词源「本含敬畏」标「一说」、不当定论。

## 总则：重庆话 ≠ 成都／四川话

成渝方言**高度共通**（同质率约 92.5%），绝大多数词标「成渝通用」即可。真正要守住「重庆味」的是下面几条**高判别度分水岭**——写错就暴露成都腔：

| 维度 | 重庆（用这个） | 成都（避开） |
|---|---|---|
| 疑问语气词「吗」 | **迈**（你吃了迈？）/ 直接「…没得？」 | 哇（你吃了哇？）—**头号避雷** |
| 「没有」 | **没得** | 莫得 |
| 「怎么」 | **啷个**（首选）；咋个/咋子也用·不禁 | —（咋个不算成都独有，见校准①） |
| 「这里／那里」 | **这点／那点**（勒里/勒点） | 这儿/那儿、这里/那里 |
| 「这个」 | **勒个** | 这个 |
| 「这么／那么」 | **恁个**（浪个） | 这们/那们、这样子 |
| 程度「很」 | **嘿X**（嘿沉，前置·优先）／得很（后置·也可） | ~惨了 |
| 骂「傻」 | **哈儿／宝器** | 瓜娃子／瓜兮兮 |
| 「加油」 | **雄起**（重庆标志喊法） | （加油） |
| 「楼下」 | 楼脚 | 楼下 |

**北方腔红线（全程禁）**：您／俺／咱们／哩、儿化卷舌（「一会儿」写「一哈／这哈」）；叹词不用「哎哟喂」，改「哦豁／哟／嗨呀」。**无「您」**：对长辈/陌生人靠**称呼**（师傅/老板儿/先生/老者/哥老倌/老师）＋语气表敬。

### 用户钦定词表（最高优先，别给相反建议）

| 普通话 | 一律写作 | 说明 |
|---|---|---|
| 尸首／尸体／尸身 | **死人子** | 角色台词统一用；X+子构词＝码头扒尸的粗野市井气，贴关二狗 |
| 昨晚／昨天晚上 | **昨晚上** | 配套：今晚＝今天黑了 |
| 句末「噻」（肯定/劝说） | **撒** | **绝非疑问**，别和迈/嗦混 |
| 句末「喃」（疑问/自问） | **葛** | ＝「呢」，本音 lán/nan |
| 丢／掉 | **落**（没掉＝没落） | 与「没得＝没有」分工：没得管「没有」、没落管「没掉」 |

| 老板儿 | 老板、掌柜 | 老板儿，行行好，赏口饭撒。 | 口语 | 此「儿」是词尾非卷舌儿化；怕混淆可写「老板」 |
| 先生 | 对读书人/算命/教书/郎中的尊称 | 先生，与我看个字撒。 | 中性/书场 | 民国通用尊称；关二狗抬举李天狗时用 |
| 哥老倌 / 老倌 | 袍哥/江湖弟兄间对成年男子的称呼 | 哥老倌些，赏个脸撒。 | 江湖 | 哥老会江湖称呼，码头帮会色彩 |
| 老师 | 对陌生人的通用尊称（无关教书） | 老师，借问一声，赶尸的从哪条路走？ | 口语 | 重庆对陌生人统一尊称，问路用 |
| 官爷 | 对官差/衙役的称呼 | 官爷，这告示上写的啥？ | 口语/年代 | 活范本用语 |

## 二、自称与骂詈（粗俗·关二狗主力区）

| 重庆词 | 普通话义 | 例句 | register | 备注 |
|---|---|---|---|---|
| 老子 | 我（粗俗自大男性自称，「你爹」口气） | 老子今天就要落你这碗饭！ | 粗俗/江湖 | 仅男性，气愤/逞凶/撑场面时上；关二狗被惹毛核心自称（活范本：「老子才不当这冤大头」「你当老子是要饭的？」） |
| 格老子 | 粗口感叹/发狠语（约「我操/我的妈呀」） | 格老子，这死人子咋个恁个沉！ | 粗俗/江湖 | 句首发语，吃惊/恼火起头（语源「本含敬畏」一说，不当定论） |
| 锤子 | 屁、个屁、才怪（强烈否定/蔑视） | 帮你？锤子哦！ | 粗俗/江湖 | **重庆标志粗口**，比成都用得更凶更随口，可多用 |
| 哈儿 | 傻子、傻瓜 | 你个哈儿，连这都看不出来。 | 粗俗 | 重庆骂傻首选（成都偏瓜娃子） |
| 哈批 / 哈麻批 | 傻子（加脏版） | —— | 粗俗 | 带「批」加重，更冲 |
| 哈（得很） | 傻、笨（形容词）；「哈得很」＝蠢得很 | 这个钱想得到？哈得很。 | 粗俗 | 活范本脚帮：「罗二爷手底下的人他能没动？哈得很。」 |
| 宝器 / 宝批龙 | 傻冒、爱出洋相又没本事 | 你个宝器，死人子身上的东西也敢拿？ | 粗俗 | 重庆高频，当面叫易翻脸（「宝气」也指娇气/小气，按语境分） |
| 龟儿 / 龟儿子 / 个龟儿 | 王八蛋；口头亲昵脏字 | 龟儿子，落了老子的钱还想跑！ | 粗俗/江湖 | 川渝通用脏话，码头口头禅，常连第二人称「你龟儿」（活范本官差：「爬爬爬！个龟儿死叫花。」） |
| 莽子 / 莽娃 | 傻大个、缺心眼的莽汉 | 那个莽子力气大，脑壳不灵光。 | 粗俗 | 「莽」也作形容词（莽起做＝闷头硬干） |
| 神戳戳 / 神兮兮 | 神经兮兮、举止反常 | 对到死人子自言自语，神戳戳的。 | 口语/粗俗 | 悬疑/诡异氛围贴；骂「傻」重庆主力仍是哈儿 |
| 戳锅漏 | 成事不足败事有余的人 | 莫喊那个戳锅漏来搭手，越帮越乱。 | 口语/贬抑 | 年代市井贬人词 |
| 背时 | 倒霉、走背运；轻骂「该死的」 | 背时哦，又遇到他！ | 口语 | ＝倒霉，也作轻骂，市井骂咧好用 |
| 涮坛子 | 开玩笑、戏耍、忽悠 | 道士莫涮老子的坛子，正经说价。 | 口语/江湖 | 江湖味玩笑词 |
| 扯把子 | 撒谎、吹牛 | 先生莫跟我扯把子，你那符到底灵不灵。 | 口语 | 市井常用 |
| 说聊斋 | 扯皮、瞎扯、说没影的话 | 莫跟我说聊斋，钱货两清。 | 口语 | 文盲关二狗说「聊斋」反有反差趣味 |
| 歪缠 | 胡搅蛮缠、纠缠不清 | 懒得跟他歪缠。 | 口语 | 活范本用语 |
| 识相 | 知趣、上道 | 行吧，算你识相。 | 口语 | 活范本官差用语 |

**关二狗骂人风格定调（重庆式）**：短、冲、劈头盖脸、嗓门大、**冲对方家属和祖上**（龟儿/老子）。**不**走成都那套绵软幽默、绕弯子的描述性长损（牙尖/占明占眼那种酸味长损）。

## 三、句末语气词与叹词

| 重庆词 | 普通话义 | 例句 | register | 备注 |
|---|---|---|---|---|
| 撒（钦定，原「噻」） | 肯定/劝说/理所当然 | 走撒，莫站起打望了。 | 口语 | 最通用肯定/劝说词，**非疑问**（活范本：「命大你可以去抓水猴子撒。」） |
| 嘛 | 理所当然/催促/停顿 | 拿都拿来了，背起就走嘛。 | 口语 | 放任语气，关二狗常用（活范本：「哄你咋子嘛？」） |
| 迈 | 疑问「吗」（是非问） | 这死人子是你扒的迈？ | 口语 | **重庆专用、对成都「哇」**。也可不用语气词直接「你吃没得？」 |
| 哈 | 叮嘱/求认同/缓和命令 | 你给我记到哈。慢点走哈。 | 口语 | 祈使句尾加哈变柔（活范本：「你看哈，这上面写的——」）。李天狗客气口吻合适 |
| 索 / 嗦（钦定优先写「索」） | 明知故问/带轻蔑反问 | 你还当真了索？／钱不想要了索？ | 口语 | 含定论的轻蔑/调侃，挑衅味（活范本脚帮：「你还当真了索？」） |
| 也 | 句末软化/陈述 | 我前几天还下水洗过澡也。 | 口语 | **研究稿漏、用户在用**：陈述句末软收，市井口语 |
| 葛（钦定，原「喃」） | 疑问「呢」（追问/自问） | 你说咋个办葛？／我落了的那把刀葛？ | 口语 | ＝「呢」 |
| 哟 | 惊讶/招呼/起头打量 | 哟，这死人子还热和的。 | 口语 | 句首起头招呼/惊讶（活范本：「哟，懂事。」「还有钱拿哟。」），句末也可拖 |
| 哦 | 升调恍然/确认；降调应答 | 哦——原来是你龟儿搞的鬼。 | 口语 | 成渝同（活范本：「好多钱哦？」） |
| 哦豁 | 事情砸了/糟了/没指望（失望无奈） | 哦豁，跑脱了！ | 口语 | **极高频重庆叹词**，出岔子最地道反应，比「完了/糟了」地道 |
| 嗨呀 / 哎呀 | 引出解释或感叹 | 嗨呀，你莫提了，今天背了一整天的死人子。 | 口语 | 起头引抱怨/解释，别带北方「哎哟」 |
| 嘿（句首叹词） | 起头招呼/喝住人 | 嘿，那个背篓的，站到起！ | 口语 | 江湖喝问味，码头混混用（区别于副词「嘿」＝很，靠位置分） |
| 咯 | 事态变化/完成（≈了） | 这下老火咯。／洗白咯。 | 口语 | 句末表变化（活范本旁白/NPC常用） |

## 四、高频动词·形容词·副词

| 重庆词 | 普通话义 | 例句 | register | 备注 |
|---|---|---|---|---|
| 晓得 | 知道、懂 | 你龟儿晓得个铲铲！ | 口语 | 西南官话核心词，否定「不晓得」；李天狗、关二狗都用 |
| 要得 | 好、行、可以（应答） | ——拖上岸。——要得，钱先讲清楚。 | 口语 | 否定/反问作「要不得」（不行） |
| 没得 | 没有 | 身上没得一个铜板。 | 口语 | **对成都「莫得」，成渝分水岭**；与钦定「没落＝没掉」不冲突 |
| 嘿（黑） | 很、非常（前置·优先） | 这具死人子嘿沉，一个人背不动。 | 口语 | **重庆最标志程度副词**，「很」读 hēi（嘿巴适/嘿吓人） |
| 得很 | 很、得不得了（后置·也用） | 码头边臭得很。／哈得很。／凶得很。 | 口语 | **成渝皆通、用户在用**（活范本：「哈得很」）；与「嘿X」并存，最重庆味优先嘿X |
| 焦（焦干/焦苦/焦黄） | 极、非常（前置，多形容坏的极端） | 那张老脸焦黄焦黄的。 | 口语 | 重庆特色程度前缀，修饰负面 |
| 稀（稀烂/稀脏） | 极、非常（前缀） | 那双草鞋遭水泡得稀烂。 | 口语 | 西南官话广用 |
| 安逸 | 舒服、爽、满意；反讽指惨 | 睡了个安稳觉，安逸惨咯。 | 口语 | **重庆比「巴适」更倾向用此**；可反讽（冷安逸了＝冷惨了） |
| 造孽 | 可怜、遭罪（西南官话特征义，非北方「作恶」） | 这娃儿爹妈都死绝了，造孽哦。 | 口语 | 常配「造孽哦」，悲悯戏好用 |
| 活路 | 活计、工作、营生、生计 | 扒死人子也是门活路，总比饿肚子强。 | 口语 | **关二狗身份本命词**，下力语境极贴（找活路/做活路） |
| 下力 / 下力人 | 出卖力气干粗活（的人）、苦力 | 我们这些下力的，命就是力气换饭。 | 口语/年代 | 民国码头阶层自称 |
| 将就 | 凑合、勉强、迁就 | 没得好的，这口薄棺材你将就用。 | 中性 | 川渝高频 |
| 撇脱 | 利索、干脆、省事 | 钱货两清，撇脱！莫啰嗦。 | 口语 | 古语词，江湖混混最爱说 |
| 默到（默倒） | 以为、还当（含想错了，mò dào） | 你默到老子怕你索？ | 口语 | 「倒/到」川渝动词后缀，年代感强 |

## 角色声口速查

### 关二狗（码头扒尸混混·粗俗文盲）
- **自称**：老子、格老子（发狠起头）；中性叙述用「我」。
- **脏字主力**：龟儿/龟儿子/个龟儿、锤子（否定）、哈儿/宝器（骂傻）、你龟儿、你娃。
- **风格**：短、冲、劈头盖脸、冲祖上；遇事「啷个搞嘛」「哦豁」「遭不住」。讨价用「好多钱」「相因」「撇脱」「钱货两清」。
- **身份词**：活路、下力、搭手、整、切、刹一脚、扒、背（死人子）。
- **句末**：撒（劝/肯定）、嘛（认命/催促）、迈（问）、索（轻蔑反问）、葛（追问）、也（软收）、哟/哦/咯。
- **绝不**：文气客套（劳驾/贫道/施主/实不相瞒/名讳…）、绕弯子长损（牙尖式）；绝不用 您/俺/咱们/儿化；绝不用 哇/莫得/巴适/瓜娃子/啦。

### 李天狗（半文游方道士）
- **该用**：莫（莫慌/莫提了）、晓得、要得、哈（缓和叮嘱：慢点走哈）、先生（互称）、造孽（悲悯）、将就。半文角色可偶用书面词稍抬。
- **风格**：客气、把话说软（多用「哈/撒」收尾），**不带粗口**——不说锤子/龟儿/老子。
- **彩蛋句**：「命在你自己手上，莫叫人替你拿了去」（碰过阿秀的本色口头禅，一字不动）。
- **绝不**：码头江湖脏话密集化；不用 您/俺/儿化；问句用迈不用哇。

### 市井NPC（茶贩、店家、街坊、围观群众、脚帮、官差）
- **该用**：师傅/老板儿/老者/婆婆/老师/官爷（称呼代敬）、晓得、要得、没得、嘛、撒、哟、哦豁、打望、安逸、造孽、年生、好多。
- **打气场面**：雄起撒！
- **风格**：生活气、年代名物（灶屋/堂屋/茅厮/棒棒/滑竿）；闲话八卦带「别个」「打望」。官差凶（爬爬爬/个龟儿/死叫花）、脚帮促狭（哄你咋子嘛/哈得很/想不得哟）。
- **绝不**：成都标志词（哇/巴适/老妞儿/楼下）；儿化/北方腔/「您」。

### 说书人（书场腔）
- **该用**：先生/哥老倌（书场称呼）、年代隐语全套（袍哥/舵把子/拜码头/吃讲茶/拉稀摆带/落教/认黄/光棍）、地理彩蛋「通远门看埋死人」「朝天门」。
- **风格**：半文半白、抑扬顿挫，可上谚语歇后语、城门民谣。
- **绝不**：现代词、普通话直译；不用 您/俺/儿化。

## 项目内活范本（用户亲笔·码头水猴子戏）

`码头看板官差.json` / `码头_脚帮帮众看板对话.json` 是用户亲手写的、口音地道的范本，凡与上文研究稿冲突，**用户实际用法是真重庆话的强证据**（已据此做上文校准①②③）。摘录定调：

- **官差（凶）**：「爬爬爬！个龟儿死叫花。下水找死，要死你死远点。」「两文？你当老子是要饭的？滚滚滚！别碍老子的眼！」「行吧，算你识相。」「哟，懂事。」
