## 背景

阿秀信号的设计文档曾分叉:旧稿把它写成"女鬼姐姐温情守护",并设"义庄=信号正式确立"的情感锚点 beat。曾有 agent 据过时稿误判出根本不存在的"确立锚点丢失 P0"。2026-06-27 用户定稿冷框架并逐处纠偏。

## 决定

阿秀信号(香粉味 + 半句跑调小调)= 阿秀死时那口**盲目的「不撒手」念气**:实物根是她帕子包里的空香粉盒(关二狗贪财捡了没舍得扔);**只认物不认人、无意识、无恨、非守护**——他快死=她的"自己"快脱手,念气盲目把他拽回。**没有单一"确立"beat**,含义靠濒死反复出现累积;demo 层只立好奇钩子,完整含义留后续支线海龟汤。最狠反讽:他吹《李天狗大战旱魃》,真相是《旱魃一路暗护关二狗》(妖护人)。

**权威定稿**:`../FindingDogStory/故事设计/阿秀支线-完整剧情与主线实施方案.md`(自标"今后创作以本文件为准";信号机理 §8b/§9,山口结局 V2 §7.5/§8/§14)。

已机制化为"死亡系绳/系统级血量"(HealthSystem + `signal_death_tether` cue,玩法说明在 `docs/玩法功能需求清单.md` §G.5;运行时细节归 runtime 域)。
