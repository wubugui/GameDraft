## 背景(一段)

规划期曾立红线"对话立绘选择必须缩略图网格弹窗、禁止长下拉";2026-07 立绘编辑器
实际落地为"三态下拉 + 表情下拉 + 72px 预览"并经实机使用,两者字面冲突。2026-07-11
首轮建库审批时制作人裁定普适边界原则,取代按控件个案立法。

## 决定(一句)

只有列表很短的枚举才允许用下拉;其它一切(大候选集、跨文件引用、视觉资产选择)
必须走弹窗选择器。

## 被否方案(防翻案)

- "立绘一律缩略图网格弹窗"按字面执行(短枚举也强制弹窗)——被普适原则取代;
  三态/表情这类短枚举下拉合规。
- 长下拉承载大候选集或引用选择——禁止;现存违规按编辑器待办逐个清理,
  不因"已经这么写了"而追认。
