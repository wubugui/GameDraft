## 背景(一段)

2026-07-05 运行时 UI 表现迭代,LibTV 生成 A/B/C/D 参考图选型,经用户两次修正收敛:要民俗草根、灰扑扑、朴素,与题材(民俗志怪)一致。

## 决定(一句)

面板观感 = **民俗草根·极简**:暖近黑底(UITheme 冷藏青已暖化)+ 一条素净旧木/墨色边 + 小圆角(3-4)近方正,别的都不要;运行时纯程序化零素材,统一经 [ui-panel-skin](../mechanisms/ui-panel-skin.md) 的单一入口铺满全部面板。

## 被否方案(防翻案)

- **雕木繁复(参考图 A)/ 满符箓(参考图 C)**:第一轮即否。
- **印章、精致金勾线、云纹角**:第二轮明确否("和民俗草根气息不搭"),云纹角贴图已删。
- **图片九宫格皮肤**:未采用,保持纯程序化。
- 字体(全项目无打包字体)是已知最大短板,**有意延后**单独做,本方向不含字体。
