## 背景(一段)

2026-07-06 对话头像系统落地后,构图经四轮用户反馈于 2026-07-07 定稿。用户核心诉求:立绘不能挡场景、服装裁切边永不可见、说话对象要有指示。

## 决定(一句)

VN 式半身像压在面板**前景**(z 序:暗幕→面板→立绘→文字),底边伸出画面底边之外,PORTRAIT_SIZE=240 的小尺寸;暗幕是 opt-in(`dimBackground=true` 才压 25% 黑);说话实体头顶挂「…」sticky 气泡与立绘并存指示说话对象。

## 被否方案(防翻案)

- **456px 大立绘**:挡场景,被否;勿再放大 240 这个尺寸。
- **默认压暗背景**:被否,压暗必须显式开。
- **立绘垫在面板之后**:脸被面板框吃掉,被否。
- **立绘底部渐隐融进面板**:被否,直接硬压前景。
