## 适用时机

任何"把一段连续内容(视频/动作/时间序列)分解成可独立取用的帧"的任务:角色动作、
环境动效、任何需要序列帧/循环的素材。不限于角色动画。本卡管**工作形状与判断**,
具体手法(选帧/锚/求解)见向下指针。

> 这是一个正交能力,与 [对抗验收抠图法](adversarial-matting.md) 正交、可各自独立使用;
> 高级任务([角色动画生产](character-animation-production.md))在两者之上组合。

## 阶段骨架(每阶段:目的 + 完成判据,不写怎么达成)

1. **定验收疆域**——目的:把"拆得合理"翻译成可判的黑白。完成判据:验收定义写死——
   ①重放忠实于源(姿态/内容逐帧等于源,不 warp);②不引入源里没有的**位置漂移/抖动**;
   ③若要循环则首尾**接缝干净**;④跨段切换不跳变(公共几何一致)。
2. **选帧 + 定公共几何**——目的:选出帧,并确定它们落进的统一坐标(尺度 + 锚)。
   完成判据:每帧落进统一坐标、内容不被裁切。手法可选(见向下指针)。
3. **对抗验收**——目的:主动去找漂移/抖动/接缝/裁切(挑最险的段),而非自证通过。
   完成判据:在正确层面度量后找不到反例(见判断点)。
4. **诊断再选**——目的:不过时定位是哪类失败再针对修,回到 2。完成判据:循环到过。

## 判断点(哪里要现场判断,拿什么证据判)

- **"稳定"的本质是稳定的锚,不是固定像素框**:源各段分辨率/取景可能不同(单一像素框物理上
  不成立);位移内容用固定框会滑出格/被裁。判据是"锚有没有把内容**逐帧平滑锁住**",
  不是"框动没动"。
- **把"引入的抖动"与"源本身的运动"分开**:验收的是有没有引入**源里没有的**漂移抖动,
  不是有没有运动。证据=静止段相邻帧的锚点位移应亚像素;运动段看重放是否忠实于源。
- **接缝**:循环段首尾相接有无跳变(自相似/端点匹配);低运动段用绝对阈值、周期动作用比值。

## 分工契约(程序做什么 / agent 裁什么 / 人拍什么板)

- **程序**:选帧、算锚、求解公共几何、算漂移/接缝候选指标、合成对照。
- **agent**:定验收疆域、对抗找最险的段(位移最猛处、跨段切换处)、裁定"引入的抖动 vs 源的运动"、决定再循环。
- **人**:拍板"这段动作该不该原地、要保留多少源运动"这类语义/玩法取舍。

## 已知死路(链 decision / 坑)

- **用固定像素框当"稳定"** → 各段分辨率不同即不成立、位移内容滑出被裁。正解=逐帧平滑锚。
- **把源本身的运动当抖动去修** → 抹掉真实动作,重放不再忠实于源。
- 选帧/锚/求解的确定性手法见向下指针,不在本卡展开。
